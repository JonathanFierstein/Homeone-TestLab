## Homeone Test Lab VM Logs Folder 

The VM Logs folder is used to stored details on many of the VMs and containers used in the lab, whether they provide network services, lab functions, or are servers and workstations that provide additional functions.

If a Test Lab node has a subfolder in the VM Logs folder, it will contain details on the nodes functions, as well as a timeline that illustrates the nodes creation, OS installation, and details on all changes from that point to the present as well as what purpose the changes have.
